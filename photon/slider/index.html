OS X Sliders

Usage

Required

The "range.js" file is not required per se. It's needed for updating the selected area. 

            <head>
  ...
  <link rel="stylesheet" href="range.css" />
  <script type="text/javascript" src="range.js"></script>
  ...
</head>

HTML Layout

            ...
  <input type="range" class="slider slider-square" />
  <!-- OR -->
  <input type="range" class="slider slider-round" />
...




Different Color

            ...
  <input type="range" class="slider slider-round blue" />
  <input type="range" class="slider slider-round gray" />
...




Different Size

            ...
  <input type="range" class="slider slider-round slider-small" />
  <input type="range" class="slider slider-round slider-bold" />
...




Squared inverted

            ...
  <input type="range" class="slider slider-square-inverted" />
...





Vertical

            ...
  <input type="range" class="slider slider-round slider-vertical" />
...

Class Reference

Level Class Name Defintion
Required 'slider' Defines input range as a slider per se
Required 'slider-square' Squared slider thumb
Required 'slider-round' Rounded slider thumb
Optional 'blue' Blue color for rounded slider thumb
Optional 'gray' Gray color for rounded slider thumb
Optional 'slider-small' Small rounded slider thumb
Optional 'slider-bold' Bold rounded slider thumb

JavaScript Documentation

There's only a prototype function 'slider.sliderVisualCalc()' should know

Use it if you append the element manually to the document, if you manually changed the "value" of the slider or if you changed something important in your layout 
            slider.sliderVisualCalc();
