Usage

Required

            <head>
  ...
  <link rel="stylesheet" href="dialog.css" />
  <script type="text/javascript" src="dialog.js"></script>
  ...
</head>

HTML Layout

            ...
  <header class="toolbar toolbar-header">
    ...
  </header>
  <div class="window-dialogs">
    ...
    <!--Here comes the dialog element-->
    <div class="dialog">
      <!--Some content for the dialog-->
      <h1 class="title">Dialog Title</h1>
      <button class="btn btn-default">Just a random button</button>
      <!--Some other dialog content, whatever you want-->
    </div>
    ...
  </div>
...

JavaScript Controlling

            //All settings are optional
dialogAction(dialogElement, {
  action: "auto", //Possible Values are "auto", "show", "hide"
  animation: true, //Can be 'true' or 'false'
  animationSpeed: 300 //Number from 0
});

Arguments

  1. Element: DOM Element
  2. Options: Object Node

Options

Name Type Values Default
action String "auto", "show", "hide" "auto"
animation Boolean true or false true
animationSpeed Number Number from 0 1000
