Circle Progress

Usage

Head

            ...
  <link rel="stylesheet" href="progress-circle.css">
  <script type="text/javascript" src="progress-circle.js"></script>
...

Body

            ...
  <!--Default the size is 20px * 20px-->
  <div class="progress-circle"></div>
  <!--You can set the size with css-->
  <div class="progress-circle" style="width: 50px; height: 50px;"></div>
...

JavaScript Reference

Get the progress

            
  var myElement = document.getElementsByClassName("progress-circle")[0];

  return myElement.progress; // Returns the progress (0 - 100);

Update the progress

            
  var myElement = document.getElementsByClassName("progress-circle")[0];

  myElement.updateCircleProgress(50); // 50%
