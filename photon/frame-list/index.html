Head

Please remeber, this package contains 3 different components:

  1. main-border Doesn't need any javascript, just CSS
  2. content-frame Doesn't need any javascript, just CSS
  3. list-select & list-editable Need javascript (and CSS of course) for controlling itself
Why there are 3 components in one?
Because the list-select or list-editable are something that only exists within an content-frame and the main-border is a so simple component that is thematically in nearby of content-frame. 

            ...
  <link rel="stylesheet" href="frame-list.css" />
  <script type="text/javascript" src="frame-list.js"></script>
...

      ...

  </div>
